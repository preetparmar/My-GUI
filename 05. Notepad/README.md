# Simple Checklist
Trying a basic GUI for a custom notepad

## Screenshot
![GUI](/05.%20Notepad/Assets/screenshot.png)

## Features:
- Create a note
- Change Font Family
- Change Font Size
- Change Font Style
- Save the file
- Square
- Open the file

## Additional Features:
- When you save the file, it saves the `Font Family`, `Font Size` and `Font Style`
